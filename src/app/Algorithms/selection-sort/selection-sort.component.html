Selection Sort

Selection Sort is one of the simplest sorting algorithms, the main issue with this algorithm is that it is slow!

View full source code here (error handling is kept to a minimum):
The algorithm can also be implemented with out creating a new list, this can found in the code with the method name: 'selectionSortInPlace'

The basic steps of the algorithm are as follows:

  1. Take an unsorted list as input
  2. Create a new empty list
  3. Search unsorted list for smallest element
  4. Add smallest element to new list
  5. Remove smallest element from old list
