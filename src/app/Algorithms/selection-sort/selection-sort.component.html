Selection Sort

Selection Sort is one of the simplest sorting algorithms, the main issue with this algorithm is that it is slow!

The algorithm moves through the list from left to right moving the smallest element into the current position. This process is carried out for every item in the list. In the code for this page a example algorithm is provided the uses the algorithm to create a new sorted list instead of sorting in place.
