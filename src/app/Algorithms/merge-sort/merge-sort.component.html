Merge Sort

Merge Sort is a sorting algorithm that is much is faster then those such as selection sort, it executes in O(n log n) time.

The algorithm involves dividing a list into individual elements and the elements into increasingly larger lists, the algorithm does not mutate the state of the original state of the list so elements of this algorithm could be run in parallel.
