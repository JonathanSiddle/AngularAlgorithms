Quick Sort

Quicksort involves selecting an item in the list (the pivot) in this implementation the last element of the list is selected. The algorithm will find the correct position for the pivot, moving items lower to the left and items greater to the right, this is called partitioning. Once the items have been partitioned, quicksort is called again with the items to the left of the pivot, then the right of the pivot.

Quick Sort is an efficient sorting algorithm and suitable to be applied to large lists.
